# WebVitals.com Tools Documentation

This directory contains documentation for the AI tools available in the WebVitals.com application.

## Available Tools

- **[Cloudflare URL Scanner](./cloudflare-scanner.md)** - Security threat analysis and malware scanning
- **[PageSpeed Insights](./pagespeed-insights.md)** - Performance analysis and Core Web Vitals metrics

## Integration

These tools are integrated into the chat interface and provide structured JSON responses that can be analyzed by the AI assistant to provide insights and recommendations to users.
